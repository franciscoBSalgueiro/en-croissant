En-Croissant — гэта адкрытая крынічная кросплатформенная шахматная GUI, якая імкнецца быць магутнай, наладжвальнай і лёгкай у выкарыстанні.


## Магчымасці

- Захоўвайце і аналізуйце свае гульні з [lichess.org](https://lichess.org) і [chess.com](https://chess.com)
- Шматматорны аналіз. Падтрымлівае ўсе рухавікі UCI
- Падрыхтоўка рэпертуару і трэніроўка з дапамогай інтэрвальнага паўтарэння
- Простая ўстаноўка і кіраванне рухавікамі і базамі дадзеных
- Пошук абсалютных або частковых пазіцый у базе дадзеных

## Зборка з крыніцы

Глядзіце [дакументацыю Tauri](https://tauri.app/v1/guides/getting-started/prerequisites) для патрабаванняў на вашай платформе.

En-Croissant выкарыстоўвае pnpm як менеджэр пакетаў для залежнасцяў. Глядзіце [інструкцыі па ўстаноўцы pnpm](https://pnpm.io/installation) для таго, як яго ўсталяваць на вашай платформе.

```bash
git clone https://github.com/franciscoBSalgueiro/en-croissant
cd en-croissant
pnpm install
pnpm build
```

Збудаваны прыклад можна знайсці ў `src-tauri/target/release`
