En-Croissant은 강력하고, 사용자 정의가 가능하며, 사용하기 쉬운 것을 목표로 하는 오픈 소스 크로스플랫폼 체스 GUI입니다.


## 기능

- [lichess.org](https://lichess.org)와 [chess.com](https://chess.com)에서 게임을 저장하고 분석하세요
- 다중 엔진 분석. 모든 UCI 엔진 지원
- 레퍼토리를 준비하고 간격 반복으로 훈련하세요
- 간단한 엔진 및 데이터베이스 설치 및 관리
- 데이터베이스에서 절대적 또는 부분적 위치 검색

## 소스에서 빌드

플랫폼의 요구사항에 대해서는 [Tauri 문서](https://tauri.app/v1/guides/getting-started/prerequisites)를 참조하세요.

En-Croissant은 의존성 패키지 관리자로 pnpm을 사용합니다. 플랫폼에 설치하는 방법에 대해서는 [pnpm 설치 지침](https://pnpm.io/installation)을 참조하세요.

```bash
git clone https://github.com/franciscoBSalgueiro/en-croissant
cd en-croissant
pnpm install
pnpm build
```

빌드된 앱은 `src-tauri/target/release`에서 찾을 수 있습니다
