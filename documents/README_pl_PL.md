En-Croissant to open-source'owy, wieloplatformowy interfejs szachowy, który ma na celu bycie potężnym, konfigurowalnym i łatwym w użyciu.


## Funkcje

- Przechowuj i analizuj swoje gry z [lichess.org](https://lichess.org) i [chess.com](https://chess.com)
- Analiza wielosilnikowa. Obsługuje wszystkie silniki UCI
- Przygotuj repertuar i trenuj go z rozłożonymi powtórkami
- Prosta instalacja i zarządzanie silnikami i bazami danych
- Wyszukiwanie pozycji absolutnych lub częściowych w bazie danych

## Budowanie z źródeł

Odwołaj się do [dokumentacji Tauri](https://tauri.app/v1/guides/getting-started/prerequisites) w sprawie wymagań na Twojej platformie.

En-Croissant używa pnpm jako menedżera pakietów dla zależności. Odwołaj się do [instrukcji instalacji pnpm](https://pnpm.io/installation), aby dowiedzieć się, jak zainstalować go na swojej platformie.

```bash
git clone https://github.com/franciscoBSalgueiro/en-croissant
cd en-croissant
pnpm install
pnpm build
```

Zbudowana aplikacja może być znaleziona w `src-tauri/target/release`
