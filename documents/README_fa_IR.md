En-Croissant یک رابط کاربری گرافیکی شطرنج چندسکویی منبع‌باز است که هدف آن قدرتمند، قابل تنظیم و آسان برای استفاده بودن است.


## ویژگی‌ها

- بازی‌های خود را از [lichess.org](https://lichess.org) و [chess.com](https://chess.com) ذخیره و تحلیل کنید
- تحلیل چند موتوره. از تمام موتورهای UCI پشتیبانی می‌کند
- یک رپرتوار آماده کنید و با تکرار فاصله‌دار تمرین کنید
- نصب و مدیریت ساده موتورها و پایگاه‌های داده
- جستجوی موقعیت‌های مطلق یا نسبی در پایگاه داده

## کامپایل از کد منبع

به [مستندات Tauri](https://tauri.app/v1/guides/getting-started/prerequisites) برای الزامات پلتفرم خود مراجعه کنید.

En-Croissant از pnpm به عنوان مدیر بسته برای وابستگی‌ها استفاده می‌کند. به [دستورالعمل‌های نصب pnpm](https://pnpm.io/installation) برای نصب آن روی پلتفرم خود نگاه کنید.

```bash
git clone https://github.com/franciscoBSalgueiro/en-croissant
cd en-croissant
pnpm install
pnpm build
```

برنامه ساخته شده را می‌توانید در `src-tauri/target/release` پیدا کنید

## اهدا

اگر می‌خواهید توسعه این رابط کاربری گرافیکی را پشتیبانی کنید، می‌توانید این کار را [اینجا](https://encroissant.org/support) انجام دهید. تمام کمک‌های مالی بسیار قدردانی می‌شوند!
