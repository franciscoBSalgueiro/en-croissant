En-Croissant, güçlü, özelleştirilebilir ve kullanımı kolay olmayı hedefleyen açık kaynaklı, çapraz platform bir satranç GUI'sidir.


## Özellikler

- [lichess.org](https://lichess.org) ve [chess.com](https://chess.com) oyunlarınızı depolayın ve analiz edin
- Çoklu motor analizi. Tüm UCI motorlarını destekler
- Bir repertuar hazırlayın ve onu aralıklı tekrarla eğitin
- Basit motor ve veritabanı kurulumu ve yönetimi
- Veritabanında mutlak veya kısmi pozisyon arama

## Kaynak koddan derleme

Platformunuzdaki gereksinimler için [Tauri belgelerine](https://tauri.app/v1/guides/getting-started/prerequisites) bakın.

En-Croissant, bağımlılıklar için paket yöneticisi olarak pnpm kullanır. Platformunuza nasıl kuracağınızı öğrenmek için [pnpm kurulum talimatlarına](https://pnpm.io/installation) bakın.

```bash
git clone https://github.com/franciscoBSalgueiro/en-croissant
cd en-croissant
pnpm install
pnpm build
```

Derlenmiş uygulama `src-tauri/target/release` içinde bulunabilir
