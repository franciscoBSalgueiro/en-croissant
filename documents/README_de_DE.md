En-Croissant ist eine quelloffene, plattformübergreifende Schach-GUI, die darauf abzielt, leistungsstark, anpassbar und benutzerfreundlich zu sein.


## Funktionen

- Speichere und analysiere deine Spiele von [lichess.org](https://lichess.org) und [chess.com](https://chess.com)
- Mehr-Engine-Analyse. Unterstützt alle UCI-Engines
- Bereite ein Repertoire vor und trainiere es mit verteiltem Üben
- Einfache Engine- und Datenbankinstallation und -verwaltung
- Suche nach absoluten oder partiellen Positionen in der Datenbank

## Aus Quellcode kompilieren

Siehe die [Tauri-Dokumentation](https://tauri.app/v1/guides/getting-started/prerequisites) für die Anforderungen auf Ihrer Plattform.

En-Croissant verwendet pnpm als Paketmanager für Abhängigkeiten. Siehe die [pnpm-Installationsanweisungen](https://pnpm.io/installation), um es auf Ihrer Plattform zu installieren.

```bash
git clone https://github.com/franciscoBSalgueiro/en-croissant
cd en-croissant
pnpm install
pnpm build
```

Die gebaute App kann unter `src-tauri/target/release` gefunden werden
