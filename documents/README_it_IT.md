En-Croissant è una GUI di scacchi open-source, multipiattaforma che mira ad essere potente, personalizzabile e facile da usare.


## Funzionalità

- Salva e analizza le tue partite da [lichess.org](https://lichess.org) e [chess.com](https://chess.com)
- Analisi multi-motore. Supporta tutti i motori UCI
- Prepara un repertorio e allenalo con ripetizione spaziata
- Installazione e gestione semplici di motori e database
- Ricerca di posizioni assolute o parziali nel database

## Compilazione dai sorgenti

Fai riferimento alla [documentazione Tauri](https://tauri.app/v1/guides/getting-started/prerequisites) per i requisiti sulla tua piattaforma.

En-Croissant usa pnpm come gestore di pacchetti per le dipendenze. Fai riferimento alle [istruzioni di installazione di pnpm](https://pnpm.io/installation) per sapere come installarlo sulla tua piattaforma.

```bash
git clone https://github.com/franciscoBSalgueiro/en-croissant
cd en-croissant
pnpm install
pnpm build
```

L'applicazione compilata può essere trovata in `src-tauri/target/release`
