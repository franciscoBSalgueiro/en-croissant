## TTS がチェス学習を変える理由

注釈付きの棋譜を復習しているとき、あなたの目は二つの仕事を同時にこなしています。盤面の駒の動きを追いながら、同時に解説文も読まなければなりません。視線は盤面と注釈パネルの間を行ったり来たりし、そのたびに一瞬だけ局面を見失います。駒の位置を探し直し、手順をたどり直し、頭の中で局面を組み立て直す必要があるのです。

テキスト読み上げ機能は、この問題を根本から解決します。

TTS を有効にすると、棋譜を進めるだけで注釈が*音声で読み上げ*られます。目は盤面に集中したままです。ナイトが f3 に着地するのを見ながら、なぜそれが優れた展開手なのかを音声で聞くことができます。ポーン構造が変化するのを見ながら、その背後にある戦略的なアイデアを解説が説明してくれます。盤面と言葉が同時に届く――まるで目の前にコーチが座って教えてくれるような感覚です。

特に以下のような場面で効果を発揮します:

- **オープニング研究** ── 局面の展開を見ながら、各手の背後にあるアイデアを耳で聞けます
- **対局の振り返り** ── 自分の注釈付き棋譜をステップ再生し、学びを自然に吸収できます
- **エンドゲーム練習** ── 重要なマスに集中したまま、解説が導いてくれます
- **語学学習との融合** ── フランス語、ドイツ語、スペイン語、ロシア語、日本語、中国語でチェスを学べます。すべてのチェス用語が自動的に翻訳されます。"Knight f3, check" の代わりに「ナイト f3、チェック」と聞こえます。あなたが考える言語でチェスを学びましょう。
- **アクセシビリティ** ── 読むよりも聴く方が楽な方や、デスクを離れて学習したい方に最適です

一度試したら、音声なしの注釈に戻ることは、映画を無音で観るような感覚になるでしょう。

## プロバイダーの選択

| **無料枠**              | 月100万文字                    | 月10,000文字                   |
| **音声品質**            | 非常に高い (WaveNet)           | 極めて高い (プレミアムAI音声)    |

**おすすめ:** まずは **Google Cloud** から始めましょう。無料枠で月100万文字まで使えます。これは数百局分の注釈付き棋譜に相当し、無料です。WaveNet の音声も素晴らしい品質です。より豊かで表現力のある、個性的な音声ナレーションが欲しくなったら、ElevenLabs をお試しください。

## Google Cloud TTS のセットアップ

この手順は約5分で完了します。Google アカウント（Gmail や YouTube で使っているもので構いません）が必要です。

### ステップ 1: Google Cloud Console にサインイン

1. ブラウザを開き、**[console.cloud.google.com](https://console.cloud.google.com/)** にアクセスします
2. Google アカウントでサインインします
3. 初めての場合、利用規約への同意を求められます。チェックボックスにチェックを入れ、**Agree and Continue** をクリックします

Google Cloud Console のダッシュボードが表示されます。画面は少し複雑に見えますが、ここで必要な操作は二つだけです。ご安心ください。

### ステップ 2: 請求先の設定

Google は無料枠であっても請求先アカウントの設定を求めます。**月100万文字を超えない限り課金されることはありません**（チェスの注釈でそこまで使うのは非常に困難です）。上限に近づくと、Google が事前に警告を表示します。

1. 画面上部の検索バーに **"Billing"** と入力し、ドロップダウンから **Billing** をクリックします
2. **Link a billing account** をクリックします（初めての場合は **Create account**）
3. 画面の指示に従い、クレジットカードまたはデビットカードを登録します
4. 完了すると、請求先アカウントの横に緑のチェックマークが表示されます

> **補足:** 別のプロジェクトで Google Cloud の請求設定を済ませている場合、このステップはスキップできます。既存の請求先アカウントがそのまま使えます。

### ステップ 3: Text-to-Speech API を有効にする

使用するサービスを Google に伝える手順です。

1. 画面上部の検索バーに **"Text-to-Speech"** と入力します
2. ドロップダウンの検索結果から **Cloud Text-to-Speech API**（青い API アイコンが目印）をクリックします
3. API の詳細ページが表示されます。大きな青い **Enable** ボタンをクリックします
4. 数秒待ちます。ボタンが **Manage** に変わったら、API が有効になっています

### ステップ 4: API key を作成する

API key は、En Croissant が Google のサーバーと通信するために使用するものです。

1. 画面上部の検索バーに **"Credentials"** と入力し、"APIs & Services" の下にある **Credentials** をクリックします
2. ページ上部付近の **+ Create Credentials** をクリックします
3. ドロップダウンから **API key** を選択します
4. ダイアログに新しいキーが表示されます。`AIzaSyC...約35文字...` のような文字列です
5. キーの横にある**コピーアイコン**をクリックして、クリップボードにコピーします
6. **Close** をクリックします

> **推奨: キーを制限しましょう。** キーを作成すると、Credentials ページに一覧表示されます。キーの名前をクリックして設定を開きます。**API restrictions** の項目で **Restrict key** を選択し、ドロップダウンから **Cloud Text-to-Speech API** を選んで **Save** をクリックします。こうすることで、万が一キーが漏洩しても、TTS 以外の用途には使えなくなります。

### ステップ 5: En Croissant の設定

あと少しで完了です！

1. En Croissant を開き、**Settings**（歯車アイコン）> **Sound** タブに移動します
2. TTS セクションまでスクロールします
3. **TTS Provider** を **Google Cloud** に設定します
4. **Google Cloud API Key** フィールドをクリックし、キーを貼り付けます（Ctrl+V）
5. **Text-to-Speech** を **On** に設定します
6. 音声選択の横にある **Test** ボタンをクリックします

チェスの手が音声で読み上げられれば、セットアップ完了です！

> **トラブルシューティング:** テスト音声が出ない場合は、次の点を確認してください。(1) API key が完全に貼り付けられているか、(2) Text-to-Speech API が有効になっているか（ステップ 3）、(3) 請求先が紐づけられているか（ステップ 2）。最もよくある原因は、API の有効化を忘れていることです。

## ElevenLabs のセットアップ

ElevenLabs はセットアップがより簡単ですが、無料枠は小さめです（月10,000文字 ── 数局分を試すには十分です）。

### ステップ 1: アカウントの作成

1. ブラウザを開き、**[elevenlabs.io](https://elevenlabs.io/)** にアクセスします
2. 右上の **Sign Up** をクリックします
3. Google、GitHub、またはメールアドレスで登録できます。お好きな方法をお選びください
4. 登録が完了すると、ElevenLabs のダッシュボードが表示されます

### ステップ 2: API key を取得する

1. ダッシュボードの左下にある**プロフィールアイコン**（または名前）をクリックします
2. **Profile + API key** をクリックします
3. API key のセクションが表示されます。**Reveal** をクリックしてキーを表示するか、まだキーがない場合は **Generate** をクリックします
4. キーは `sk_...約30文字...` のような文字列です
5. **コピーアイコン**をクリックして、クリップボードにコピーします

### ステップ 3: En Croissant の設定

1. En Croissant を開き、**Settings**（歯車アイコン）> **Sound** タブに移動します
2. TTS セクションまでスクロールします
3. **TTS Provider** を **ElevenLabs** に設定します
4. **ElevenLabs API Key** フィールドをクリックし、キーを貼り付けます（Ctrl+V）
5. **TTS Voice** ドロップダウンに利用可能な音声が表示されます。**Adam** は優れたデフォルト選択です。明瞭で自然な音声で、チェスの解説にぴったりです
6. **Text-to-Speech** を **On** に設定します
7. 音声選択の横にある **Test** ボタンをクリックします

チェスの手が音声で読み上げられるはずです。

> **無料枠について:** ElevenLabs の無料プランでは月10,000文字まで使えます。注釈付きの棋譜1局あたり2,000〜4,000文字程度を使用するため、月に2〜5局分を無料で復習できます。TTS が有用だと感じたら、Starter プラン（月額$5、30,000文字）がおすすめのアップグレードです。Pro プラン（月額$22、100,000文字）はヘビーユーザー向けです。

## 設定リファレンス

すべての TTS 設定は **Settings > Sound** にあります:

| 設定項目                   | 機能                                                                           |
|--------------------------|-------------------------------------------------------------------------------|
| **Text-to-Speech**       | すべての TTS 機能の有効/無効を切り替えるメインスイッチ                                |
| **Auto-Narrate on Move** | 矢印キーで手を進めたときに注釈を自動的に読み上げます                                  |
| **TTS Provider**         | ElevenLabs と Google Cloud を切り替えます                                        |
| **ElevenLabs API Key**   | ElevenLabs の API key（ElevenLabs を使用する場合のみ必要）                         |
| **Google Cloud API Key** | Google Cloud の API key（Google を使用する場合のみ必要）                            |
| **TTS Voice**            | ElevenLabs: お好みの音声を選択。Google: 男性または女性を選択                         |
| **TTS Language**         | ナレーションの言語 ── すべてのチェス用語が自動的に翻訳されます                         |
| **TTS Volume**           | ナレーションの音量                                                               |
| **TTS Speed**            | 再生速度（0.5倍〜2倍） ── 音声の再生成なしに即座に調整されます                        |
| **TTS Audio Cache**      | キャッシュされた音声をクリアして再生成を強制します（注釈を編集した後に便利です）           |

## 対応言語

TTS ナレーションは現在、チェス用語が完全に翻訳された7言語に対応しています:

| Language           | Chess example                                      |
|--------------------|-----------------------------------------------------|
| **English**        | Knight f3, check. A strong developing move.         |
| **Francais**       | Cavalier f3, echec. Un coup de developpement fort.  |
| **Espanol**        | Caballo f3, jaque. Un fuerte movimiento.            |
| **Deutsch**        | Springer f3, Schach. Ein starker Entwicklungszug.   |
| **日本語**          | ナイト f3、チェック。強い展開の手。                      |
| **Русский**        | Конь f3, шах. Сильный развивающий ход.              |
| **中文**            | 马 f3，将军。一步控制中心的强力出子。                    |

駒の名前、「チェック」「チェックメイト」「キャスリング」「取る」、さらに「素晴らしい手」「悪手」といった手の評価 ── すべてのチェス用語が選択した言語で読み上げられます。PGN ファイル内のコメントはそのまま読み上げられるため、聞きたい言語で棋譜に注釈を付けてください。

## より良い体験のためのヒント

- **Auto-Narrate を活用しましょう。** "Auto-Narrate on Move" をオンにして、矢印キーで棋譜を進めるだけです。手を進めるたびに解説が自然に流れてきます。まるで肩越しにコーチがいるかのような感覚です。

- **自分の対局に注釈を付けましょう。** TTS が真価を発揮するのは、*自分の*対局の解説を聴くときです。対局に注釈を付けてから、ナレーション付きでステップ再生してみてください。局面を見つめながら「このポーンを取るのは魅力的に見えますが、キングサイドはまだ全く展開できていません」と聞こえてくる体験は、テキストで読むのとはまるで違います。

- **再生速度を試してみましょう。** じっくり研究したい方は1倍速、素早く復習したい方は1.3倍速を好む傾向があります。速度スライダーはリアルタイムで再生速度を調整し、追加の API 文字数は消費しません。音声は一度だけ生成され、再生速度のみが変わります。

- **スピーカーアイコンを活用しましょう。** 手順リストの各コメントには小さなスピーカーアイコンが付いています。クリックすると、棋譜全体を進めることなく、その注釈だけを聴くことができます。

- **言語を切り替えてチェス用語を学びましょう。** 第二言語でチェスを学んでいるなら、TTS の言語をそれに合わせてみてください。聴いているだけで、"Cavalier"（ナイト）、"echec"（チェック）、"mat"（チェックメイト）といった用語が自然と身につきます。

## この機能について

En Croissant は、[Francisco Salgueiro](https://github.com/franciscoBSalgueiro) が開発したオープンソースのチェス学習ツールです。Francisco は本当に素晴らしいものを作り上げました。無料で高機能な、コミュニティ主導のチェス学習プラットフォームを GPL-3.0 ライセンスで公開し、誰もが使い、改良し、共有できるようにしたのです。この TTS 機能が存在するのは、その寛大さのおかげです。彼が築いた基盤に感謝するとともに、このプロジェクトに貢献できることを誇りに思います。

TTS プラグインは、[Red Shed](mailto:darrell@redshed.ai) の Darrell が [Claude Code](https://claude.ai/claude-code) の助けを借りて開発しました。多言語対応、デュアルプロバイダー統合、7言語にわたるチェス用語の翻訳 ── すべてソースからビルドし、手作業でテストし、心を込めて貢献しています。

これがオープンソースの醍醐味です。誰かが素晴らしいものを作る。別の誰かがそれに機能を加える。そして、みんなが恩恵を受ける。
